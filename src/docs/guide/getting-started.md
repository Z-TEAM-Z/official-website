## 环境准备

- Node.js >= 24.14.1
- pnpm >= 11

## 安装依赖

```bash
pnpm install
```

## 启动开发服务器

```bash
pnpm dev
```

访问 `http://localhost:4321` 即可查看项目。

## 构建生产版本

```bash
pnpm build
pnpm preview
```
